# Profile Photo Instructions

## Adding Your Photo

1. Place your profile photo in this directory: `/public/images/`
2. Rename it to: `profile.jpg` or `profile.jpeg`
3. Recommended specifications:
   - Size: 400x400px to 800x800px (square format)
   - Format: JPG or PNG
   - File size: Under 500KB for optimal loading

## Current Setup

The portfolio is configured to display your photo in the hero section next to your name and title.

If the photo is named differently, update the image path in `src/App.tsx` at line ~245.
